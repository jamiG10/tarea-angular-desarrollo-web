Tarea de Desarrollo Web

Autora: Jamileth Gongora

Esta tarea consiste en practicar los conceptos de Angular y conectarse a una API pública. El objetivo es fortalecer las habilidades en el desarrollo de aplicaciones web utilizando Angular, manejando solicitudes HTTP y procesando datos de una API externa para mejorar la interacción con fuentes de información.

Proyecto realizado para la asignatura de Desarrollo Web.
